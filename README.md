# 🍇 FruitHub – Fruit Company Management System

FruitHub is a comprehensive **company management system** designed for businesses operating in the fruit distribution and sales domain. The system focuses on real-world operational workflows such as inventory, orders, employees, payroll, and customer management.

---

## 📌 Notes

* This repository was created specifically for **testing and analysis purposes**.
* The project was previously pushed to a different repository by a teammate, where multiple web applications were stored together, resulting in unnecessary size and redundancy.
* For the course **INT3117 (Class INT3117_1)**, this repository isolates **FruitHub** as a standalone application to be used for **load and performance testing with the Locust testing tool**.
* No functional changes were made to the core application logic; this separation is solely for clarity, maintainability, and testing scope.

## ✨ Key Features

### 📦 Product & Supplier Management

* Add, search, and filter products
* Track out-of-stock items
* Manage suppliers and supplier-based filtering

### 📊 Orders & Revenue

* Track order statuses (processing, completed, cancelled)
* View order details and total values
* Filter orders by date range
* Calculate revenue (overall or by period)
* Identify top customers by purchase volume

### 👥 Customer Management

* Store detailed customer profiles
* View purchase history
* Identify loyal customers
* Track unpaid debts and payment history

### 🧑‍💼 Employee Management

* Employee profiles with role-based permissions
* Search by department, ID, or name
* Add/remove employees
* Track employee health and family information

### 💰 Payroll & Attendance

* Attendance tracking (late check-ins, absences)
* Salary calculation with penalties and bonuses
* Department- and individual-based bonuses
* Total payroll cost calculation

### ⭐ Customer Feedback

* Manage ratings and feedback
* Filter positive/negative reviews
* Identify poorly rated orders
